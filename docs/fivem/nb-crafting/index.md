# nb-crafting

Sistema de crafteo para FiveM (ESX/QBCore). Lo que puedes tocar: **config.lua** y, si usas bandas, **bridge** para Origen Ilegal.

---

## Lo importante

- **[Instalación](instalacion.md)** — Requisitos, base de datos, config mínima y arranque.
- **[Configuración](configuracion.md)** — Opciones de config.lua y textos/idiomas.
- **[Origen Ilegal (bandas)](origen-ilegal.md)** — Cómo hacer que las restricciones de estaciones y recetas usen la **banda** del jugador en lugar del job. Solo hay que cambiar 6 funciones en `bridge/framework.lua` (o cargar el archivo alternativo); en la doc están los bloques para copiar y pegar.

El resto del recurso está encriptado. Framework e inventario se eligen en config; no hace falta tocar código salvo para Origen Ilegal.
