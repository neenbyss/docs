# Comandos

Todos los comandos requieren permisos de admin.

| Comando | Descripción |
|---------|-------------|
| `/showcars` | Abrir menú de vehículos |
| `/givecar [id] [modelo]` | Entregar vehículo a un jugador |
| `/delcar [matricula]` | Eliminar vehículo de la base de datos |
| `/owncar` | Registrar vehículo actual como propio |
| `/setupwelcomecar` | Obtener coordenadas para posicionar el NPC |

## Permisos

Por defecto, los comandos requieren:
- **ESX**: grupo `admin` o `superadmin`
- **QBCore/QBox**: permiso `admin` o `god`

Consulta la sección de [Configuración](configuracion.md) para usar ACE Permissions.
