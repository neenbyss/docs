# nb-cars

Sistema de gestión de vehículos todo-en-uno para FiveM. La forma más sencilla de administrar vehículos en tu servidor.

## Características

- 🎫 **Sistema Completo**: Gestiona vehículos de jugadores (ver, spawnear, transferir, eliminar)
- 🎁 **Welcome Car**: Sistema de NPC para que nuevos jugadores reclamen un vehículo de bienvenida
- 🔑 **Multi-llaves**: Integración con Brutal Keys, QB-VehicleKeys, QS-VehicleKeys, CD Garage, Wasabi Carlock, Jaksam o sistema personalizado
- 🌐 **Multi-framework**: Compatible con ESX, QBCore y QBox
- 🔧 **Exports**: Úsalo desde otros scripts para dar vehículos, consultar BD o gestionar llaves
- 🛡️ **Permisos Flexibles**: Usa rangos de framework o ACE permissions
- 🌎 **i18n**: Todos los textos configurables

## Compatibilidad

- ESX (Legacy y 1.2+)
- QBCore
- QBox
- oxmysql
- ox_lib
