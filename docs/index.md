## Cómo usar esta documentación

- Usa el **menú lateral** o las **pestañas** para saltar entre categorías.
- El **buscador** (lupa) encuentra términos en todas las páginas.
- Cada sección tiene un índice con enlaces a cada proyecto o guía.

## Añadir nueva documentación

Si trabajas en este repo: crea un `.md` en la carpeta correspondiente (`minecraft/` o `fivem/`) y añade la entrada en `mkdocs.yml` dentro de la sección `nav` adecuada.
